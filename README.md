# Hotel Admin Dashboard

A modern admin dashboard for managing hotel operations, including rooms, reservations, users, and bookings.

## Preview

<!-- ![Admin Dashboard Screenshot](./public/screenshot.png) -->

## Features

- 🔐 Admin authentication
- 🏨 Manage hotel rooms
- 📅 Manage reservations and bookings
- 👤 Manage users
- 📊 Dashboard statistics
- 🔍 Global search for rooms, users, and reservations
- 📝 Form validation and error handling
- 📱 Responsive design

## Tech Stack

- React
- Vite
- React Router
- TanStack React Query
- React Hook Form
- Tailwind CSS
- React Hot Toast

## API Integration

This dashboard communicates with the backend API to:

- Fetch hotel data
- Manage rooms
- Handle reservations
- Manage users

Server state is handled using TanStack React Query.

## Installation

Clone the repository:

git clone https://github.com/username/hotel-admin-dashboard.git

Install dependencies:

npm install

Run development server:

npm run dev
